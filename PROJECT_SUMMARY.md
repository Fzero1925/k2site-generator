## 📋 使用流程

1. **初始化项目**
   ```bash
   k2site init my-website
   cd my-website
   pnpm install
   ```

2. **生成内容**
   ```bash
   k2site generate -k "React教程" "Vue.js入门" "前端开发"
   ```

3. **本地开发**
   ```bash
   k2site dev
   ```

4. **构建部署**
   ```bash
   k2site build
   k2site deploy --target cloudflare
   ```
